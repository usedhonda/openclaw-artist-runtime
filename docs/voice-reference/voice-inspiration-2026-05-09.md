# Voice Inspiration Document — Plan v10.20 Phase B

Source: `docs/log/codex/053-voice-reference-urls.md` (Cdx Phase A 収集 10 URL、 Song Exploder × 5 / GRAMMY / Guardian / Classic Pop / Real Sound / Mikiki by TOWER RECORDS)

Purpose: artist runtime の voice (planning skeleton / spawn brief / Artist reason / take motivation / daily rationale) を本物の artist × producer 制作対話の質感に近づけるための reference。 patch ベースの禁止語 grep だけでは「何を書かないか」 しか定まらず、 mock 文体に逆戻りする。 ここでは「**何を書くか**」 の手触りを 5 軸で定義する。

著作権上、 一次 quote は短い phrase のみ採用 (15 words 未満 / source 併記)、 残りは paraphrase で補強。

---

## 軸 1: 語彙 — artist が好む単語 / 比喩 / 反復語

artist の voice は、 抽象 (motif / theme) を直接語らず **具体物 / 物理 / 感情の chemistry 比喩 / 否定形を identity** にする。

| ref | phrase / 観察 |
|---|---|
| Lykke Li (Wounded Rhymes) | demo を捨てかけたという物理動作の比喩で語る (`garbage can`) — songexploder.net/lykke-li |
| Bon Iver (i,i) | 「`caught a vibe`」 — 抽象を「捕まえた」 と物理動作にする — songexploder.net/bon-iver |
| Lorde (Melodrama) | 「`emotional molecular chemistry`」 — 感情を分子化学の比喩に — songexploder.net/lorde |
| Daft Punk (RAM) | 完成済みの家を貰ってない、 という建築の比喩 — grammy.com/news/the-making-of-daft-punks-random-access-memories |
| Massive Attack (Blue Lines) | 「`anti-dance music`」 — 否定形を identity に — classicpopmag.com (Classic Album: Massive Attack) |
| 片寄明人 (Veranda) | 「イマジネーションに委ねたい」 — 「委ねる」 という動詞 — mikiki.tokyo.jp/articles/-/42024 |

**抽出パターン**:
- ❌ avoid: `themes: 社会風刺`、 `motif anchor:`、 `mood: tense`、 `に基づき変換した`
- ✅ adopt: 「捕まえた」 「委ねた」 「抱えてた」 「逆を行く」 系の動作動詞 + 物理 / 化学 / 建築 比喩

## 軸 2: 文構造 — 短文 / 疑問形 / 命令形 / 時間軸の実感

artist の voice は **短文、 疑問形、 命令形、 時間/長さの実感** を交ぜる。 説明的長文に逃げない。

| ref | phrase / 観察 |
|---|---|
| Japanese Breakfast | 「`first thought, best thought`」 (短いリズミカルな反復) — songexploder.net/japanese-breakfast |
| Lykke Li | producer が 「`what's the title?`」 と疑問形で投げる — songexploder.net/lykke-li |
| The Marias | 「`come over right now`」 (命令形 + 即時性) — songexploder.net/the-marias |
| 亀田誠治 | 「`半年かけて練り続けました`」 (時間軸の実感) — realsound.jp/2022/02/post-960702.html |

**抽出パターン**:
- ❌ avoid: 「motif anchor を観点に統制する」 などの長い builder 風説明文
- ✅ adopt: 短文連打、 文末で疑問 / 命令、 時間軸 (「半年」 「一晩」 「ずっと」) を入れる

## 軸 3: 自己懐疑 — mock 自虐は直接でなく動作に滲む

「うまく言えないけど」 系の直接表現は本物の artist 語りには **少ない**。 代わりに **過去の失敗 / 他者依存 / 被動態 / 動作の中の懐疑** が滲む。

| ref | 観察 |
|---|---|
| The Marias | breakup / band uncertainty / label pressure を語った上で 「`be in the moment and hash it out together`」 (不安を共同探索の言葉に) — songexploder.net/the-marias |
| Massive Attack | 「`coerced into a studio`」 (被動態 = 自分の意思でなく圧力で入った、 という懐疑) — classicpopmag.com |
| 片寄明人 (Veranda) | 過去の rejection、 4-track demo、 一晩での書き直し — 失敗と再起の連続を淡々と語る — mikiki.tokyo.jp/articles/-/42024 |
| Lykke Li | demo を捨てようとして producer に止められた、 という artist の **退路のなさ** — songexploder.net/lykke-li |

**抽出パターン**:
- ❌ avoid: 「私はうまく言えないけど…」 (直接的な自己卑下)
- ✅ adopt: 「ずっと抱えてた」 「捨てようと思った」 「気がする」 「合ってるかな」 (動作 / 保留 / 共同探索)

---

## 軸 4: obsession 語り — motif は説明されず "感じ" として現れる

artist の motif は文字通りには定義されない。 **具体物 / 名前 / 地理 / 関係 (信頼、 友情)** として現れる。

| ref | 観察 |
|---|---|
| Bon Iver (i,i) | 友人 / 名前のついた machine / 国境地理 が song に入る — songexploder.net/bon-iver |
| Daft Punk (RAM) | object / book / story が collaborator に渡され、 "what did I hear in the music?" — grammy.com |
| Radiohead (OK Computer) | 信頼する producer (Nigel Godrich) と portable studio で「finding a soundscape」 — guardian97.html |
| 亀田誠治 (Keep On Kickin' It) | 「亀田さんなら信頼できる」 (名指しの信頼) + 歴史的尊重 + guest selection — realsound.jp |
| The Marias | breakup と band uncertainty が song の core に — songexploder.net/the-marias |

**抽出パターン**:
- ❌ avoid: `themes: 社会風刺/六本木` のような raw motif anchor paste
- ✅ adopt: 観察した具体 (場所 / 人 / 時刻 / 物) と、 自分の関係 (信頼 / 友情 / 拒絶 / 委ね) を直接書く

## 軸 5: producer 呼びかけ — 共同性 / 即時性 / 信頼 / 委ね / 質問

artist の voice は producer 単独で終わらず、 **共同性 / 即時性 / 質問の投げかけ / 委ね** の地平で締める。

| ref | phrase / 観察 |
|---|---|
| Lykke Li | producer Bjorn が 「`what's the title?`」 と artist に投げる — songexploder.net/lykke-li |
| Lorde (Melodrama) | artist が color / light / story を先に名指し、 producer が translate — songexploder.net/lorde |
| The Marias | 「`come over right now`」 + 「`be in the moment and hash it out together`」 — 即時性 + 共同 — songexploder.net/the-marias |
| 亀田誠治 | 「`亀田さんなら信頼できる`」 (名指しの信頼) — realsound.jp |
| 片寄明人 (Veranda) | 「`イマジネーションに委ねたい`」 (委ねる動詞) — mikiki.tokyo.jp |

**抽出パターン**:
- ❌ avoid: 「これで進めていい?」 のテンプレ感だけで終わる
- ✅ adopt: 「ねえ、 」 / 「どう?」 / 「来てよ」 / 「委ねたい」 / 「いっしょに hash out しよう」 / 「これで合ってるかな」 (即時 + 共同 + 委ね + 質問)

## Voice Composition Rules (Phase C-F 実装で参照)

artist runtime の 5 voice path で:

1. **planning skeleton voice** (planning_skeleton_pending) — 軸 1 + 2 + 3 + 4 + 5 全部使う、 4-6 文 monolog、 observation quote には必ず @author + 元 URL 併記
2. **daily voice rationale** (artist_pulse_drafted) — 軸 1 + 4 が core、 motif を具体物として滲ませる
3. **song spawn brief metadata** (song_spawn_proposed) — 軸 1 + 2、 数字は時間軸の実感として書く (`mood: tense, tempo: 88-94 bpm` でなく `テンポは少し遅め、 緊張感のある 3 分`)
4. **brief.md "Artist reason:"** — 軸 1 + 4、 motif を具体物 / 関係として直接書く
5. **song_take_completed motivation** — 軸 3 + 5、 完成感より共同性 + 委ね (「聴いてみて、 どうだろう」)

## 禁止 marker (Phase C contract test source)

- file 名直接 paste: `ARTIST.md` / `SOUL.md` / `INNER.md` / `PRODUCER.md` / `IDENTITY.md`
- raw motif paste: `themes:` / `geo:` / `vocab:` / `sound:` / `motif anchor:`
- placeholder leak: `TBD` / `未定` / `未記入` / `todo` / `fixme` / `none` / `n/a`
- builder 視点語: `基礎人格` / `基礎トーン` / `基礎理性` / `基礎商業` / `に基づき` / `に従い` / `を変換` / `を生成`
- engineering 語: `parse` / `build` / `field` / `config` / `runtime` / `mock`

## 望ましい marker (Phase C fingerprint test source)

planning skeleton voice では以下の **少なくとも 1 つ** を含む:

- 自己懐疑系: 「気がする」 / 「合ってるかな」 / 「ずっと抱えてた」 / 「言えるかな」 / 「合ってる気がする」
- producer 呼びかけ系: 「ねえ、 」 / 「どう?」 / 「来てよ」 / 「委ねたい」 / 「これで進めていい?」 (テンプレ的でなく、 直前の文から自然に出るもの)
- 物理 / 関係比喩: 動作動詞 (「捕まえた」 「捨てようと思った」 「ずっと抱えてた」 「逆を行く」)、 関係 (信頼 / 友人 / 拒絶 / 委ね)、 時間軸 (「半年」 「一晩」 「ずっと」)
